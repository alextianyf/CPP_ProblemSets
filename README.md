# CPP_ProblemSets

This is a CPP problem sets for learning purpose. Please do not use this CPP problem sets for any commercial purpose.

- IDE: VsCode

- Common command:
  - Creating directory

        mkdir "directory name"
        <br/>

  - Changing directory

        cd path
        <br/>

  - Creating file

        code fileName.cpp
        <br/>

  - Compile and generate .exe file

        g++ fileName.cpp -o Generated.exe
        <br/>

  - Run .exe file

        ./Generated.exe

<br/>
